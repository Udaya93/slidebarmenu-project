# slidebarmenu-project
CSS Sidebar Menu Project
This project implements a sleek and modern sidebar menu using HTML, CSS, and FontAwesome icons. 
The sidebar is hidden by default and can be toggled by clicking on a menu button. The sidebar includes navigation links and social media icons.

Features
1.Responsive Design: The sidebar menu is designed to be responsive and works well on various screen sizes.
2.Smooth Transition: The menu slides in and out smoothly using CSS transitions.
3.FontAwesome Icons: Utilizes FontAwesome icons for a visually appealing interface.
4.Custom Fonts: Uses the Poppins font from Google Fonts for a clean and modern look.


Files
index.html: The main HTML file containing the structure of the sidebar menu.
slidebarmenu.css: The CSS file that styles the sidebar menu.
